# Unicorns Custom Seeds
Unicorns Custom Seeds allows you to synthesize your custom weed mixes into seeds via a new quest with Albert Hoover. The mod features visual matching for vials and plants, Botanist support, and persistence across save files. Find your new seeds in shops or deaddrops and enjoy a seamless growing experience.

**This mod is not compatible with the Mod Manager Phone App by Prowiler**

# What does this mod do?
- Create seeds for your custom weed mixes via Albert Hoover
- Add a new color pallet to weed plants
- Improves automation of weed mixes by creating Weed Seeds for discovered mixes
- Persistence for each Save Game
- Configurable using melon preferences or [Mods App by k0Mods](https://thunderstore.io/c/schedule-i/p/k0Mods/ModsApp/)

# Installation
1. Download and install [MelonLoader](https://github.com/LavaGang/MelonLoader/releases)
1. Download [S1API](https://github.com/ifBars/S1API/releases) and follow the setup steps
1. Download this mod.
1. Unzip the downloaded file
1. Make sure you select the correct mod for the branch of the game you have installed. Il2Cpp for main branch & Mono for alternate branch.
1. Drag the .dll into the "Mods" folder in your Schedule 1 files.
